## Description

GraphiQL is a playground/IDE for GraphQL queries. Our application contains the following structure:

- Welcome page
- User authorization
- GraphiQL page with:
  - Query editor
  - Variable editor
  - Title editor (optional)
  - Documentation Explorer (should be downloaded with a delay)
  - Answer section
